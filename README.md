# FREEL - Gestion Freelance Intelligente

Application web de pilotage d'activite pour micro-entrepreneurs et freelances.

## Fonctionnalites

- **Dashboard** - Vue d'ensemble avec indicateurs cles (CA, charges, tresorerie)
- **Missions** - Gestion des projets et suivi client
- **Factures** - Creation et registre des factures
- **Charges** - Suivi des depenses professionnelles
- **Tresorerie** - Gestion des flux de tresorerie
- **Simulateur fiscal** - Calculs URSSAF, IR, TVA

## Utilisation

Ouvrir `index.html` dans un navigateur ou acceder a :
**https://loisgagne11-dot.github.io/FREEL/**

## Caracteristiques techniques

- Application 100% front-end (pas de serveur requis)
- Stockage local (localStorage)
- PWA installable
- Mode hors-ligne
- Export PDF des factures
